## Getting Started

To use this repository:

1. Clone the repository

```bash
git clone https://github.com/dareen-abualhaj/nextflows-internship.git
```

2. Open the project folder

```bash
cd nextflows-internship
```

3. Pull the latest updates

```bash
git pull
```
